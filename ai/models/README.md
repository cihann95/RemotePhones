# AI/ML Models for Phone Farm System

This directory contains machine learning models used for:
- App usage prediction
- Device behavior anomaly detection
- Screen content analysis
- Automation optimization

## Model Types

1. **Usage Prediction Models** - Predict when apps will be launched
2. **Anomaly Detection Models** - Identify unusual device behavior
3. **Image Classification Models** - Analyze screen content for automation
4. **Recommendation Models** - Suggest optimal automation sequences

## Training

Models can be trained using the scripts in `ai/training/`.

## Inference

Use the inference utilities in `ai/inference/` to load and run models.
