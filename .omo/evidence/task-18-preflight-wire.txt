Design Decisions:
- Preflight runs BEFORE createWindow() so errors can be caught before UI loads
- Uses dialog.showMessageBox (not IPC) since window doesn't exist yet when errors occur
- Warnings are logged-only; only errors trigger the dialog
- ensureEnvFile() is already called inside runPreflightChecks(), no separate call needed
- All Turkish error messages and fix steps come from preflight.js (Task 4)
