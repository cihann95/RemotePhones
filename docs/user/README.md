# Phone Farm Documentation

Phone Farm is a platform for managing Android phones from your computer via ADB. You can connect multiple devices, run automated tasks, make phone calls, and monitor everything from a desktop GUI.

## Getting Started

Start here if you are new to Phone Farm.

- [Getting Started](getting-started.md): Prerequisites, installation, and first device connection.
- [Setup Wizard](setup-wizard.md): First launch of the Electron GUI, selecting Home or Office mode.

## Device Management

- [Device Management](device-management.md): Connect phones, discover devices, run health checks, and manage tasks.
- [Phone Calls](phone-calls.md): Make single or bulk calls, answer, hang up, and monitor call state.

## Reference

- [CLI Reference](cli-reference.md): All command-line commands with examples.
- [Glossary](glossary.md): Definitions of ADB, USB Debugging, Scrcpy, and other terms.
- [Troubleshooting](troubleshooting.md): Solutions for common problems.
